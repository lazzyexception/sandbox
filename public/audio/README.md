# Optional birthday soundtrack 🎵

Drop an audio file here named **`soundtrack.mp3`** to enable the floating music
control (it stays **off by default** — the visitor chooses to turn it on).

```
public/audio/soundtrack.mp3
```

The brief's song was *"I Love u"* — use whatever you have the right to use.

To use a different filename or format, change `soundtrack` in
`src/data/birthdayConfig.ts`. Set it to `""` to hide the sound control entirely.
